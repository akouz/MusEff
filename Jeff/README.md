# JEFF овердрайв

JEFF - это двухкаскадный [JFET овердрайв](https://github.com/akouz/MusEff/blob/main/Assymetrical/README.md#2-%D1%83%D1%81%D0%B8%D0%BB%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9-%D0%BA%D0%B0%D1%81%D0%BA%D0%B0%D0%B4-%D0%BD%D0%B0-jfet), подобный однокаскадному овердрайву [Полярная звезда](https://github.com/akouz/MusEff/blob/main/Polaris/README.md). При разработке Jeff преследовались две цели:

-- сравнить звучание одного и двух каскадов каскодного ассиметричного [JFET клиппера](https://github.com/akouz/MusEff/blob/main/Assymetrical/README.md#2-%D1%83%D1%81%D0%B8%D0%BB%D0%B8%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9-%D0%BA%D0%B0%D1%81%D0%BA%D0%B0%D0%B4-%D0%BD%D0%B0-jfet)

-- избавиться от TC1044, заменив его копеечным генератором на триггере Шмитта TC4S584

![Jeff](https://raw.githubusercontent.com/akouz/MusEff/refs/heads/main/Jeff/Jeff_rev_2_1.png)
