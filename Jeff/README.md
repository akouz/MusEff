# JEFF овердрайв

JEFF - это двухкаскадный JFET овердрайв, подобный однокаскадному овердрайву Полярная звезда. При разработке Jeff преследовались две цели:

-- сравнить звучание одного и двух каскадов каскодного ассиметричного JFET клиппера

-- избавиться от TC1044, заменив его копеечным генератором на триггере Шмитта TC4S584

![Jeff](https://raw.githubusercontent.com/akouz/MusEff/refs/heads/main/Jeff/Jeff_rev_2_1.png)

Пример звучания, Сампл, который при помощи Audiacity был:

-- нормализован

-- сведен в моно

-- обработан ФНЧ 4-го порядка с частотой среза 6 кГц (ака аналог абстрактного гитарного динамика)

В примере риф играется 4 раза:

1. GAIN 9:00, один каскад

2. GAIN 9:00, два каскада

3. GAIN 15:00, один каскад

4. GAIN 15:00, два каскада

Остальные ручки в среднем положении (т.е. 12:00)
