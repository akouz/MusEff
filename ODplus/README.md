# Овердрайв с ВЧ-клиппером

Клиппер с подмесом ВЧ сигнала описан [здесь](https://github.com/akouz/MusEff/tree/main/Clippers). Клиппирующие диоды D3 - Шоттки.  ВЧ генератор треугольного напряжения частотой не менее 100 кГц собран на ОУ U3A. Пассивный ФНЧ второго порядка R13C16R16C17 с частотой среза 7 кГц подавляет ВЧ в выходном сигнале до уровня, достаточного для нормальной работы ОУ U2B. Окончательное подавление ВЧ происходит в регуляторе тембра R19R22С22. 

Переключатель XS1 позволяет сравнить звучание с ВЧ сигналом и без него. Без подмеса ВЧ эффект звучит как довольно обычный, не очень жесткий овердрайв на Шоттки диодах. В нем слышится небольшой песок. Когда подмес ВЧ включен, песок полностью исчезает, звучание очищается и начинает напоминать звучание лампового овердрайва.
