# Овердрайв с ВЧ-клиппером

Клиппер с подмесом ВЧ сигнала описан [здесь](https://github.com/akouz/MusEff/tree/main/Clippers). Клиппирующие диоды D3 - Шоттки.  ВЧ генератор треугольного напряжения частотой не менее 100 кГц собран на ОУ U3A. Пассивный ФНЧ второго порядка с частотой среза 7 кГц подавляет ВЧ в выходном сигнале.
