 # Polar Bear

 Ячейка Гилберта лежит в основе аналоговых умножителей и транскондакторных усилителей. В этом овердрайве используется предельно упрощенная ячейка. Сигнал подается на базу транзистора Q1A и одновременно на эмиттер транзистора Q2A. В результате на коллекторе транзистора Q1A получается умноженный сам на себя, т.е. возведенный во вторую степень сигнал. Если на вход подается синусоидальный сигнал, то на коллекторе транзистора Q1A формируется сигнал удвоенной частоты. На коллекторе Q1B формируется смесь входного сигнала и сигнала, возведенного в квадрат.

Для малых сигналов искажения пропорциональны амплитуде входного сигнала. Для получения бОльших искажений две ячейки соединены последовательно. Выходной сигнал первой ячейки ослабляется делителем R16,R17 так, чтобы уровень сигнала на входе второй ячейки был равен уровню сигнала на входе первой. Тогда обе ячейки вносят одинаковый вклад в искажения.

Звучание напоминает трехкаскадный ламповый усилитель. В представленном примере регулятор усиления находится в положении 9:00 половину времени и в положении 15:00 другую половину. Сигнал был записан на линейный вход и не подвергался обработке.
