# Умножитель

Из тригонометрии известно, что возведение в квадрат функции синуса даст функцию удвоенной частоты:

sin2(x) = (1- cos(2x))/2          [1]

Поскольку в звуке синус и косинус на слух неотличимы, для простоты можно сказать, что возведение в квадрат удваивает частоту сигнала. Это наглядно видно на графике фиг. 1. Пропустив синусоидальный сигнал (зеленая линия) через квадратичную передаточную функцию (красная парабола) мы на выходе получим сигнал удвоенной частоты (фиолетовая линия).
