# Овердрайв с ассиметричным диодным клиппером

Ассимметричный диодный клиппер был рассмотрен [здесь.](https://github.com/akouz/MusEff/tree/main/Assymetrical#readme) В представленной на Fig.1 схеме используется двухкаскадный клиппер. Первый каскад состоит из резистора R6 и диодов D2,D3. Второй каскад состоит из резистора R9 и диода D4. Положительное смещение на клиппер формируется диодом D6 и настраивается подстроечником R16.
