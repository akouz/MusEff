# Polaris

The core is a [JFET in cascode configuration](https://github.com/akouz/MusEff/blob/main/Assymetrical/README_en.md#2-jfet-amplifier-stage). The signal spectrum of this stage is virtually free of harmonics above the second until hard clipping begins.

The operating mode is set with trimmer pot R11. Hard clipping with LEDs is optional, depending on taste. I liked it better without the LEDs.

The circuit consisting of Schottky diode D7 and resistor R23 is also optional. It simulates the effect of the tube's grid currents and adds some liveliness to the sound.
