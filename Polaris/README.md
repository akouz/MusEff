# Полярная Звезда

Основу составляет JFET в каскодном включении. В спектре сигнала такого каскада вплоть до начала жесткого клиппирования практически отсутствуют гармоники выше второй.

Режим работы задается подстроечником R11. Жесткое клиппирование светодиодами опциональное, по вкусу. Мне без светодиодов больше понравилось.

Цепочка из диода Шоттки D7 и резистора R23 тоже опциональная. Она имитирует влияние сеточных токов лампы и дает некоторое оживление звучания.
