# Sweet Honey Overdrive

Использована такая же идея, как в овердрайве Sweet Honey фирмы Mad Professor: клиппер, охваченный обратной связью. Регулируя глубину ОС, можно регулировать искажения, вносимые клиппером.

Правда, на уровне идеи сходство и заканчивается. Использованы два клиппера со смещением, включенные последовательно. Первый клиппер охвачен ОС, глубина которой регулируется подстроечником R8. Последовательно включенные клипперы с мягкой характеристикой позволяют просуммировать вносимые ими искажения и сделать звук богаче.
